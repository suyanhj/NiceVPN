# Proposal: VPN 管理后台 bug 文档闭环（2026-04）

## 背景

依据 `app/bug文档.md` 对 OpenVPN 管理后台进行一轮集中修复与增强，涵盖初始化、用户/组、防火墙、证书与服务可见性。

## 目标

- 初始化：全局 CIDR 任意合法前缀、PKI 生成完整服务端证书、默认组默认放行规则、实例写入配置。
- 组管理：子网校验语义清晰、禁止删除默认组、列表展示完整组 ID。
- 用户：批量创建带后缀、模糊搜索、设备绑定重置、列表信息单行化与简易流量可视化。
- 防火墙：全量实例规则重建（修复多 owner 覆盖）、ipset+iptables 源匹配、空条件下列出全部规则、侧栏表单、多端口与拖拽顺序保持。
- 证书：`easyrsa` 路径与续签后用户序列号同步。

## 非目标

- 不改变现有 JSON 存储根目录结构；不引入外部数据库。
